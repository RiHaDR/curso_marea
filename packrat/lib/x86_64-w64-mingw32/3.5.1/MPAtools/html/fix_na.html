fix_na {MPAtools}R Documentation

Fix NA

Description

Removes all na-variables (see value) to avoid errors when performing the linear model

Usage

fix_na(data)

Arguments

data

a data.frame, generated by any of the MPAtools::indicator functions

Value

a data.frame where all-na variables (i.e. the entire column had NA) are replaced with a 1
