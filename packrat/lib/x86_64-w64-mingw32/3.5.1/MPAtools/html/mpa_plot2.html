mpa_plot2 {MPAtools}R Documentation

Plots results from biophysical indicator functions

Description

Plots results from the functions used to calculate biophysical indicators like: richness(), density(), ttrophic(), fish_biomass(), and fish_size().

Usage

mpa_plot2(data, error.bars = F, y.lab = "Indicador")

Arguments

data

dataframe generated by any of the following functions: richness(), density(), trophic(), fish_biomass(), and fish_size()

error.bars

A logical value indicating if error bars (1 SD) should be plotted.
